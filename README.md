# easy_web_view

Easy Web Views in Flutter on Web and Mobile!

## Getting Started

Setup iOS Info.plist

```
<key>io.flutter.embedded_views_preview</key>
<true/>
```

For Loading a new url or changing width/height just call setState!

```dart
EasyWebView(
  src: 'https://flutter.dev',
// width: 100,
// height: 100,
)
```
